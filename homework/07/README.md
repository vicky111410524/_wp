# JavaScript 全方位實作挑戰（1～10題）

## 📌 專案簡介
本作業為 JavaScript 基礎到進階概念練習，涵蓋物件操作、陣列處理、Callback、JSON、Template Literals 以及模擬後端資料流程等內容。

主要目的在於熟悉：
- 前端資料處理
- Callback 運作方式
- Node.js 常見資料結構模式
- 模擬 API / 資料庫行為

---

## 🧠 題目內容與重點

---

### 1. 物件屬性存取
學習使用 dot notation 與 bracket notation 存取物件資料。

---

### 2. 物件解構賦值
從物件中快速取出指定欄位（常見於 API 回傳資料處理）。

---

### 3. 陣列遍歷與 HTML 拼接
使用 `forEach` 將資料轉成 HTML 字串。

---

### 4. 動態物件屬性
模擬 URL params（如 `/post/99`）。

---

### 5. Callback 函數傳參
模擬非同步資料回傳（Error-first callback 基礎概念）。

---

### 6. JSON 解析
將 JSON 字串轉換為 JavaScript 物件並存取資料。

---

### 7. 模擬資料庫查詢
模擬 `db.get()` 的行為與 callback 回傳機制。

---

### 8. Template Literals 條件判斷
使用模板字串進行動態 HTML 生成。

---

### 9. 字串處理與截斷
模擬文章摘要功能（substring + ...）。

---

### 10. Error-first Callback
學習 Node.js 常見錯誤優先回呼模式。

---

## 🚀 技術重點整理

- Object（物件操作）
- Array methods（map / forEach / filter）
- Callback function
- JSON.parse()
- Template Literals (\`\`)
- 模擬後端 API 行為
- Error-first callback pattern

---

## 🧪 執行方式

請使用 Node.js 執行：

```bash
node index.js
````

或直接貼到瀏覽器 Console 測試。

## 📊 學習成果

完成本作業後，你將能理解：

* JavaScript 資料流動方式
* 前後端資料交換概念
* Callback 在實務中的用途
* 基本 API / DB 模擬邏輯
* 動態網頁內容生成方式
